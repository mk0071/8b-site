# 8B Ultra Premium Final — 13 Languages

Финальная премиальная версия сайта 8B с 13 языками:
Русский, English, Español, Français, Deutsch, 中文, العربية, हिन्दी, Português, Türkçe, Bahasa Indonesia, 日本語, 한국어.

## Как обновить сайт
В GitHub замени 4 файла: index.html, styles.css, app.js, README.md.
После Commit changes Vercel обновит сайт автоматически.

## Перед запуском оплаты
В app.js замени CONFIG.TREASURY на адрес кошелька 8B Treasury.
